Software Design & Engineering

Restaurant Mobile App Enhancement - SQLite integration, cart functionality implementation, and comprehensive feature expansion planning

About the Artifact

The artifact I selected is a mobile restaurant information app created in the CS 360 course, "Mobile Architecture and Programming." It was built using React Native and Firebase. The app originally allowed users to view restaurant lists, see menu items, and log in. For CS 499, I successfully enhanced this application by implementing SQLite database integration, cart functionality, and developing comprehensive plans for additional feature expansions.

I chose this artifact because it demonstrates my ability to create user-friendly mobile applications and integrate them with robust backend systems. The enhancement process showcases both practical implementation skills and strategic software design planning.

Successfully Implemented Enhancements

SQLite Database Integration

Successfully migrated the application from Firebase to a comprehensive SQLite database system featuring 9 interconnected tables: Users, Restaurants, Categories, Menu_Items, Orders, Order_Items, Cart, Favorites, and Reviews. This implementation provides local data persistence and eliminates external dependencies.

Secure Authentication System

Implemented a robust authentication system with password hashing and secure login functionality. The system ensures user data protection while maintaining seamless user experience across app sessions.

Shopping Cart Functionality

Developed and integrated shopping cart functionality that allows users to add menu items, manage quantities, and persist cart data. The cart system demonstrates successful implementation of complex state management and database integration.

Planned Feature Expansion Design

Reservation System Architecture

Designed a comprehensive reservation system architecture that would allow users to:

  • Select restaurants from the main listing with improved navigation
  • Choose preferred date and time slots through intuitive interface design
  • Specify party size and special requests with form validation
  • Submit reservations with comprehensive confirmation workflows
  • View and manage existing reservations through dedicated interface

Review Submission System Design

Planned functionality for comprehensive user review system including star rating interface, text review input with validation, photo upload capability planning, and review display system with sorting and filtering options.

Enhanced User Experience Planning

Developed detailed wireframes and user journey maps for the planned features, ensuring seamless integration with existing functionality while maintaining consistent design patterns throughout the application.
