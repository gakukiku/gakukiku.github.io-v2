Core Computer Science Skills

Working in Team Environments

Throughout my computer science program, I developed essential teamwork skills through various team-based projects and iterative development processes. During CS 499, I experienced meaningful collaboration through the milestone feedback system, where I incorporated instructor guidance to improve my database enhancement from a basic SQLite implementation to a sophisticated system with proper organization and security features. This iterative process taught me the value of receiving constructive criticism and adapting my approach based on expert feedback.

Communicating with Stakeholders

My ability to communicate complex technical concepts to diverse audiences was significantly strengthened through writing comprehensive reports for each artifact enhancement. When documenting my database migration from Firebase to SQLite, I learned to explain technical decisions in terms of business value—describing not just what I implemented, but why these changes matter for performance, security, and maintainability. This skill of translating technical complexity into clear business benefits will be crucial in my future role as an IT infrastructure consultant.

Data Structures and Algorithms

My enhancement of a Binary Search Tree program into an AVL Tree implementation demonstrated my deep understanding of algorithmic efficiency and performance optimization. This project required me to analyze the limitations of basic BST operations and implement rotation algorithms to maintain O(log n) performance guarantees. Through this work, I gained practical experience with complex recursive algorithms and learned to measure and validate performance improvements through systematic testing.

Software Engineering and Database

My most comprehensive demonstration of software engineering expertise came through completely redesigning my restaurant app's data architecture. I transformed a simple Firebase-connected application into a sophisticated SQLite-based system featuring nine interconnected tables: Users, Restaurants, Menu_Items, Orders, Cart, Favorites, Categories, Order_Items, and Reviews. This enhancement required applying database normalization principles to eliminate data redundancy while maintaining data integrity through carefully designed relationships.

Security

Developing a security mindset has become integral to my approach to software development, particularly evident in my database enhancement project where I implemented comprehensive security measures. I replaced simple text-based authentication with a robust system featuring password hashing and salt generation, ensuring that user credentials are protected even if the database is compromised. Additionally, I implemented parameterized queries throughout the application to prevent SQL injection attacks, demonstrating my understanding that security must be built into the foundation of any data-driven application.
